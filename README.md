Dotfiles
========

* dotfiles - python cli tool to automate symlinking dotfiles

Install
-------

```bash
pip install dotfiles
git clone --recursive -j8 git@github.com:nfrigus/dotfiles.git
cd dotfiles
dotfiles --sync
```
